Hemp Tracker

Traceability of hemp production using blockchain technology.

The hemp industry, both recreational and medicinal, is gaining ground again in the world as more countries join in the decriminalization and regulation of the use of these plants. The proposal of Hemp Tracker is to help hemp producers, who want to enter to compete in this market, to improve the transparency of the production process and bring all that information to the final consumer, digitally, with a simple QR on the product packaging itself.

My rol in this project was to design the landing page for the presentation of the application. It was made using Angular.

Lovebets

Register your love on blockchain.

Lovebets is a project that allows you to register your marriage on the Telos blockchain, and bet that you will be together until death do you part. The smart contract code is written in C++, the tests are written in Python and the UI was made using Vue framework.
